Strategy Logic
📈 BUY Signal triggers when:
EMA 9 crosses above EMA 21
RSI(14) below 65 (not overbought)
Volume > 1.5× 20-bar average
📉 SELL Signal triggers when:
EMA 9 crosses below EMA 21
RSI(14) above 35 (not oversold)
Volume > 1.5× 20-bar average
🛡️ Stop-Loss:
ATR(14) × 1.5 below entry (BUY)
Falls back to fixed % if ATR unavailable
Monitored every 30 seconds
📡 Data Source:
Binance public API (free)
Fallback: KuCoin public API
No API key required
